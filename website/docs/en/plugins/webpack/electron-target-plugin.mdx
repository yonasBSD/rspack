# ElectronTargetPlugin

This plugin is used to external the Electron built-in modules during bundling, and is used by [`externalsPresets.electron`](/config/externals#externalspresetselectron), [`externalsPresets.electronMain`](/config/externals#externalspresetselectronmain), [`externalsPresets.electronRenderer`](/config/externals#externalspresetselectronrenderer), and [`externalsPresets.electronPreload`](/config/externals#externalspresetselectronpreload) under the hood.

```js
new rspack.electron.ElectronTargetPlugin();
```
