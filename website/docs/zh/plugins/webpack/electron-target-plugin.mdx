# ElectronTargetPlugin

这个插件用于在打包时 external Electron 内置模块，被 [`externalsPresets.electron`](/config/externals#externalspresetselectron)、[`externalsPresets.electronMain`](/config/externals#externalspresetselectronmain)、[`externalsPresets.electronRenderer`](/config/externals#externalspresetselectronrenderer) 和 [`externalsPresets.electronPreload`](/config/externals#externalspresetselectronpreload) 所使用。

```js
new rspack.electron.ElectronTargetPlugin();
```
